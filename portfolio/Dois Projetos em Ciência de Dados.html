1 Introdução

Nesse relatório, apresento dois scripts feitos por mim ao longo da duração do curso, descrevendo os procedimentos realizados em cada etapa. O primeiro analisa os dados da Pesquisa Nacional de Saúde de 2019 e apresenta resultados de uma regressão linear a partir deles. O segundo realiza uma análise de séries temporais com dados em painel sobre criminalidade e desemprego.

O relatório foi inteiramente construído em Quarto, permitindo a execução de código em R e sintetizando tudo em um documento único.

2.2 Carregamento e tratamento dos dados

O script lê a base de dados da Pesquisa Nacional de Saúde (PNS) 2019:

pns_2019 <- read.csv2("dados_pns/PNS_2019.csv")

Depois, ele filtra os dados para incluir apenas respondentes do questionário individual:

pns_2019_red <- pns_2019[which(
  pns_2019$V0015 == 1 & pns_2019$M001 == 1 & pns_2019$M00203 == 1), ]

Em seguida, são selecionadas apenas as variáveis de interesse, e essas são devidamente renomeadas.

2.3 Transformações de variáveis

Diversas variáveis são ajustadas para melhor representação nos modelos:

  • Saúde subjetiva geral: escala invertida para manter a coerência com a interpretação dos coeficientes.
  • Atividades sociais: também ajustadas para garantir consistência.
  • Variáveis binárias (dummies): criadas para sexo e raça.
  • Densidade domiciliar: calculada como o número total de moradores dividido pelo número de cômodos da residência.
## (Des)inverter variável de saúde
pns_2019_red$saude_subjetiva_geral <- 6 - pns_2019_red$saude_subjetiva_geral

## (Des)inverter variáveis de atividades
pns_2019_red$atividade_civica <- 7 - pns_2019_red$atividade_civica
pns_2019_red$atividade_voluntaria <- 7 - pns_2019_red$atividade_voluntaria
pns_2019_red$atividade_religiosa <- 7 - pns_2019_red$atividade_religiosa

## Criar variável de atividades sociais (média)
pns_2019_red$nivel_atividade_social <- rowMeans(pns_2019_red[, c("atividade_civica", "atividade_voluntaria", "atividade_religiosa")], na.rm = TRUE)


## Transformar sexo em dummy
pns_2019_red$sexo_feminino <- ifelse(pns_2019_red$sexo == 2, 1, 0)

## Idade quadrada
pns_2019_red$idade2 <- pns_2019_red$idade^2

## Criar variável de densidade domiciliar
pns_2019_red$densidade_domiciliar <- pns_2019_red$total_moradores/pns_2019_red$comodos_domicilio

## Transformar raça em dummy
pns_2019_red$raca_branca <- ifelse(pns_2019_red$raca == 1, 1, 0)
pns_2019_red$raca_preta <- ifelse(pns_2019_red$raca == 2, 1, 0)
pns_2019_red$raca_amarela <- ifelse(pns_2019_red$raca == 3, 1, 0)
pns_2019_red$raca_parda <- ifelse(pns_2019_red$raca == 4, 1, 0)
pns_2019_red$raca_indigena <- ifelse(pns_2019_red$raca == 5, 1, 0)
pns_2019_red$raca_ignorada <- ifelse(pns_2019_red$raca == 9, 1, 0)

  "Sul" = c(41, 42, 43),

  p_valor_amigos = numeric(),
  stringsAsFactors = FALSE

Para rodar as regressões, cria-se um loop que repete as regressões para cada região.

# Loop para realizar duas regressões por região
for (regiao in names(regioes)) {
  # Filtrar os dados para a região atual
  estados_da_regiao <- regioes[[regiao]]
  dados_regiao <- subset(pns_2019_red, UF %in% estados_da_regiao)
  
  # Regressão 1: Familia
  modelo_familia <- ivreg(
    saude_subjetiva_geral ~ familia + sexo_feminino + idade + idade2 +
      raca_preta + raca_amarela + raca_parda +
      raca_indigena + raca_ignorada + educacao + faixas_rendapc +
      densidade_domiciliar |
      total_moradores + sexo_feminino + idade + idade2 +
      raca_preta + raca_amarela + raca_parda +
      raca_indigena + raca_ignorada + educacao + faixas_rendapc +
      densidade_domiciliar,
    data = dados_regiao
  )
  
  # Regressão 2: Amigos
  modelo_amigos <- ivreg(
    saude_subjetiva_geral ~ amigos + sexo_feminino + idade + idade2 +
      raca_preta + raca_amarela + raca_parda +
      raca_indigena + raca_ignorada + educacao + faixas_rendapc +
      densidade_domiciliar |
      nivel_atividade_social + sexo_feminino + idade + idade2 +
      raca_preta + raca_amarela + raca_parda +
      raca_indigena + raca_ignorada + educacao + faixas_rendapc +
      densidade_domiciliar,
    data = dados_regiao
  )
  
  # Extrair coeficientes e p-valores
  coef_familia <- coef(summary(modelo_familia))["familia", "Estimate"]
  p_valor_familia <- coef(summary(modelo_familia))["familia", "Pr(>|t|)"]
  
  coef_amigos <- coef(summary(modelo_amigos))["amigos", "Estimate"]
  p_valor_amigos <- coef(summary(modelo_amigos))["amigos", "Pr(>|t|)"]
  
  # Adicionar os resultados ao dataframe
  resultados_coefs <- rbind(
    resultados_coefs, 
    data.frame(
      regiao = regiao,
      coef_familia = coef_familia,
      p_valor_familia = p_valor_familia,
      coef_amigos = coef_amigos,
      p_valor_amigos = p_valor_amigos
    )
  )
}

3.2.1 Dados de criminalidade

A base é organizada para transformar categorias de crimes em colunas separadas:

Os nomes das variáveis são renomeados para facilitar a leitura:

names(crime_ts) <- c("country", "year", "burglary", "theft")

Apenas países que possuem dados completos de 2003 a 2022 são mantidos:

years_needed <- 2003:2022

crime_ts <- crime_ts %>%
  group_by(country) %>%
  filter(all(years_needed %in% year)) %>%
  ungroup()

Alguns nomes de países são ajustados para garantir uniformidade:

crime_ts <- crime_ts %>%
  mutate(country = case_when(
    country == "Netherlands (Kingdom of the)" ~ "Netherlands",
    country == "Republic of Korea" ~ "South Korea",
    TRUE ~ country

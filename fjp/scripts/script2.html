Introdução

Criei esse arquivo em RMarkdown para explicar melhor o meu raciocínio ao longo da execução do código e para deixá-lo mais apresentável para vocês.

Obs.: Para rodar esse script, basta adicionar o CSV “informacoesLivros_skoob” do 1º Script que enviei para vocês na pasta desse arquivo, ajustar o caminho de referência no segundo bloco de código e rodar o script.

O primeiro passo para realizar a análise dos dados é a limpeza da base.

Começarei pela coluna “genero”. Nela, há todos os gêneros nos quais um livro se enquadra. Essa classificação é feita através de strings que seguem o formato “Genero Primário / Gênero Secundário / Gênero Terciário…”. Podemos quebrar essa coluna em categorias separadas, para melhor observação.

# Separar a coluna "genero" em 8 outras colunas (utilizando tidyr)
base_livros <- separate(base_livros, genero, into = c("genero1", "genero2", "genero3", "genero4", "genero5", "genero6", "genero7", "genero8"), sep = " / ")

# Exibir base ajustada (utilizando knitr)

Agora, vamos para a coluna “local_autor”: nela, há as informações de país, estado e cidade de nascimento do autor de cada livro. Para as nossas análises, só o país é relevante, então modificaremos as strings para que elas contenham apenas essa informação, possibilitando agrupar as observações em categorias maiores.

# Pegar apenas os países da coluna "local_autor" (utilizando stringr)
delimitador = ' \n'

base_livros$local_autor <- str_extract(base_livros$local_autor, paste0(".*(?=", delimitador, ")"))

# Vamos também eliminar os diferentes nomes da coluna "local_autor"
base_livros[base_livros$local_autor == "Estados Unidos da América", "local_autor"] <- "Estados Unidos"
base_livros[base_livros$local_autor == "EUA", "local_autor"] <- "Estados Unidos"
base_livros[base_livros$local_autor == "Califórnia", "local_autor"] <- "Estados Unidos"

base_livros[base_livros$local_autor == "Inglaterra", "local_autor"] <- "Reino Unido"
base_livros[base_livros$local_autor == "Grã-Bretanha", "local_autor"] <- "Reino Unido"

Outra possibilidade de tratamento é identificada: vemos que algumas variáveis dessa base não podem ser utilizadas por serem do tipo “chr”. Vamos convertê-las para o tipo “dbl”, com o qual podemos fazer operações matemáticas.

# Substituir os símbolos de porcento de todos os elementos das colunas 21 a 27 (utilizando dplyr e pipe)
base_livros[, 21:27] <- base_livros[, 21:27] %>%
  mutate_all(list(~str_replace(., '%', '')))

# Retirar todos os pontos dos números (utilizando dplyr e pipe)
base_livros[, c(13:18, 20)] <- base_livros[, c(13:18, 20)] %>%
  mutate_all(list(~str_replace(., '\\.', '')))

# Remover string "avaliações" do final de todos os elementos da coluna "num_avaliacoes"
base_livros$num_avaliacoes <- gsub('.{11}$', '', base_livros$num_avaliacoes)

# Agora, só precisamos transformar todas as colunas modificadas em variáveis numéricas
base_livros[, 13:27] <- sapply(base_livros[, 13:27], as.numeric)

Visualização dos dados

Por fim, apresento 5 visualizações dos dados tratados nesse script. Foquei nas análises mais relevantes dos dados, que permitissem imagens mais interessantes e que utilizassem mais recursos da biblioteca “ggplot2”.

Obs.: Nas visualizações onde são apresentadas regressões, coloco conjuntamente os coeficientes da referida regressão, para referência.

1ª Visualização: Gráfico de barras - Número de leituras vs. País de origem do autor

# Criar dataframe-resumo, com o somatório de leituras por país (utilizando dplyr e pipe)
a <- base_livros %>%
  group_by(local_autor) %>%
  summarise(sum(status_leram))

# Alterar nome das colunas
colnames(a) <- c("local_autor", "status_leram")

# Montar gráfico (utilizando ggplot2)
ggplot(a, aes(x = reorder(local_autor, status_leram), y = status_leram, fill = status_leram))+
  geom_bar(stat="identity") +
  coord_flip() +
  scale_fill_gradient(low="blue",high="darkblue") +
  labs(title = "Número de Leituras por País", x = "Países de origem dos autores", y = "Número de leituras") +
  scale_y_continuous(labels = unit_format(unit = "M", scale = 1e-6)) +
  theme(panel.border = element_blank(), panel.grid.major = element_blank(),
panel.grid.minor = element_blank(), axis.line = element_line(colour = "black"), legend.position = "none")
